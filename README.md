**COMMANDS**                                                                                     
                                                                                     
/pinsystem reload - Reload configuration.                                                                                     
/pinsystem version - Plugin version.                                                                                     
                                                                                     
/pinsystem change <player> <newPin> - Change player Pin.                                                                                     
/pinsystem delete <player> - Reset player pin.                                                                                     
/pinsystem clearcache - Clear player cache.                                                                                     

**PERMISSIONS**
